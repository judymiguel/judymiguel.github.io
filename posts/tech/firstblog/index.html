一、构建图像分类数据集#

注：如果都在云平台运行，不需要B4了

1.制作图像分类数据的注意事项#

数据集应尽可能包括目标物体的各类场景，训练出的图像分类模型才能在各类测试场景中具备好的泛化性能，防止过拟合。#

不同尺寸、比例的图像 不同拍摄环境（光照、设备、拍摄角度、遮挡、远近、大小） 不同形态（完整西瓜、切瓣西瓜、切块西瓜） 不同部位（全瓜、瓜皮、瓜瓤、瓜子） 不同时期（瓜秧、小瓜、大瓜） 不同背景（人物、菜地、抠图） 不同图像域（照片、漫画、剪贴画、油画）

关于构建数据集的学习中有许多注意点，数据集的质量影响着模型最后分类的效果，这一点确实很重要。#

如果出现账户未知情况，如下图#

Git提交需要有贡献者的信息，所以有两种方案

  1. 设置全局用户名和邮箱 git config –global user email “邮箱” git config –global user email “贡献者名称”
  2. 为单个项目设置独立用户名和邮箱 在项目根目录下输入不加global的命令 git config user email “邮箱” git config user email “贡献者名称”
